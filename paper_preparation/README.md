# 论文准备文档使用指南

## 📚 文档列表

本文件夹包含4个核心文档，用于论文写作的准备工作：

1. **01_related_work.md** - 相关工作文献综述
2. **02_methodology.md** - 方法章节草稿
3. **03_experiments.md** - 实验设计表格
4. **04_results_analysis.md** - 结果分析模板

## 📖 文档说明

### 1. 相关工作文献综述 (01_related_work.md)

**内容概要：**
- 7个研究方向：眼底图像分析、多任务学习、自监督学习、病灶感知学习、医学影像分析、跨域泛化、研究Gap总结
- 20篇核心论文（必读10篇 + 重要10篇）
- 技术对比表（模型架构、预训练方法、多任务学习方法）
- 文献阅读计划（4周计划）
- BibTeX格式的参考文献

**使用方法：**
1. 按照阅读计划，每周精读3-5篇论文
2. 填充表格中的具体数值和细节
3. 根据实际阅读情况更新论文列表
4. 整理BibTeX参考文献

**预计时间：** 4周（每周5-10小时）

---

### 2. 方法章节草稿 (02_methodology.md)

**内容概要：**
- 总体框架设计（4个核心模块）
- 共享视觉编码器（RetFound ViT-Large）
- 任务特定预测头（ODIR和DDR）
- 病灶感知模块（3种方法）
- 跨任务对齐模块（2种方法）
- 训练策略（三阶段训练）
- 实现细节（框架、硬件、超参数）
- 创新点总结

**使用方法：**
1. 根据实际实现调整架构图和公式
2. 更新超参数和训练策略
3. 补充具体的实现细节
4. 根据实验结果调整创新点表述

**预计时间：** 2-3周（与代码实现同步）

---

### 3. 实验设计表格 (03_experiments.md)

**内容概要：**
- 主实验设计（ODIR、DDR、跨数据集）
- 对比方法设计（11个基线方法）
- 消融实验设计（5组消融实验）
- 统计显著性检验设计
- 实验执行计划（4个阶段）
- 结果记录表格（4个表格）

**使用方法：**
1. 根据实际资源调整实验配置
2. 按照执行计划逐步完成实验
3. 实时填写结果记录表格
4. 根据实验结果调整实验设计

**预计时间：** 6-8周（实验执行）

---

### 4. 结果分析模板 (04_results_analysis.md)

**内容概要：**
- 结果分析框架（6个分析维度）
- 性能提升分析（整体、类别、少数类）
- 消融实验分析（组件贡献、损失权重、训练效率）
- 错误分析（混淆矩阵、失败案例、错误统计）
- 可解释性分析（注意力图、病灶定位、临床评估）
- 泛化能力分析（跨数据集、跨质量、跨子群）
- 统计显著性分析（t-test、Wilcoxon、置信区间）
- 结论和建议（主要结论、局限性、未来工作）
- 分析代码模板（4个Python函数）

**使用方法：**
1. 收集实验结果后，按照模板进行分析
2. 使用提供的代码模板生成图表
3. 填充所有表格中的数值
4. 根据分析结果撰写结论

**预计时间：** 2-3周（结果收集后）

## 💡 使用建议

### 1. 灵活调整
- 根据实际进度调整时间安排
- 根据实验结果调整实验设计
- 根据反馈修改文档内容

### 2. 迭代优化
- 文档不是一成不变的，需要持续更新
- 每完成一个阶段，回顾并完善之前的文档
- 根据实验结果调整方法章节

### 3. 版本控制
- 使用Git管理文档版本
- 定期备份重要文档
- 记录每次修改的内容和原因

### 4. 协作建议
- 如果有合作者，明确分工
- 定期同步进度和文档
- 使用共享文档平台（如Google Docs）
